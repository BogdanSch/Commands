Работа в консоли

ls - вывести список содержимого папки, в которой находимся

cd - сменить папку

mkdir - создать папку

touch - создать любой файл

code - создать файл для написания в нём кода

rm –rf - удалить папку или файл

Работа с Git и GitHUB

  • git init - инициализировать репозиторий
  • git status - показывует текущий статус
  • git add - отслеживает изминения файлов
  • git commit - сохраняет изминения в коммите
  • git config - конфигурирует параметры git
  • git branch - работа с ветками в репозиториия
  • git checkout - переключается га другую ветку
  • git merge - совмещает текущую ветку с выбранной
  • git pull
  • git clone - клонирует проект с удалённого репозитория
